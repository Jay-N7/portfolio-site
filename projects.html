Click Fraud Detection System

Python • Machine Learning

Built an end-to-end machine learning pipeline for click fraud detection (cybersecurity / ad-tech), using a 1000+row train dataset and 500+row test dataset with 56 features, framed as a binary classification problem. Performed data preprocessing + feature engineering including missing value imputation, removal of ID-like features (~10+ columns), and One-Hot Encoding for categorical variables, improving model readiness and feature quality. Trained and benchmarked multiple models (Logistic Regression, Random Forest, Gradient Boosting) using scikit-learn, evaluated with accuracy, precision, recall, F1-score, ROC-AUC, addressing class imbalance in fraud detection. Achieved best performance with Random Forest (Accuracy: 91.71%, Precision: 0.6667, Recall: 0.6087, F1-score: 0.6364); implemented modular ML pipeline (data → training → evaluation → prediction).

AI-Powered Log Intelligence (ONGOING)

Kafka • FastAPI • Elasticsearch • ML

Designing a distributed log ingestion pipeline using Apache Kafka and Elasticsearch to process real-time system and security logs from microservices architecture, enabling scalable observability, monitoring, and security event analysis. Implementing machine learning-based anomaly and threat detection using Isolation Forest and LSTM models on time-series logs to identify system failures, performance degradation, and potential security anomalies / suspicious behavior patterns. Developing an automated root cause analysis (RCA) module using clustering techniques to correlate log events, supporting faster identification of incident sources, system faults, and potential security incidents in distributed environments. Building a FastAPI-based auto-remediation engine to trigger predefined recovery and incident response workflows upon anomaly detection, targeting reduced MTTR (Mean Time To Recovery), improved system resilience, and automated incident response handling.
